## d2ds-dslings | 练习

通过使用[xlings](https://github.com/d2learn/xlings)下载项目并运行自动化检测的**编译器驱动开发模式**来进行代码练习

> 注: [xlings安装文档](https://github.com/d2learn/xlings)

### 获取练习代码

```bash
xlings drepo d2ds
```

### 执行dslings进入练习

> 进入d2ds目录, 执行dslings开始练习

```bash
cd d2ds
xlings dslings
```

### 打开电子书

```bash
xlings book
```
